{{var this.message}}
If a template has a provider, the data of that provider can be accessed by referencing it with this. For eg,

Lets say we have a template file hello.html which has a data provider data.json. The content of the data.json file is as follows, 

{{var this.sample_data_provider}}

So in the template file, we can access this data as follows, 

<h1>{{var this.heading}}</h1>
<p>{{var this.message}}</p>

The output of the following code would be,

{{var this.sample_data_provider.heading}}

{{var this.sample_data_provider.message}}
