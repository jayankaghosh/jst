The JST engine comes with a built-in preprocessor capable of executing inline JS in your templates. So for example, lets say we have an array we want to loop through and display. We can either use the data-loop attribute, or we can use the JST preprocessor to loop inside the template itself.

For example, lets say we have a JSON which looks something like this,


{{var JSON.stringify(this)}}

Using the preprocessor, inside the template we can write something like,


<table class="table">
	<tr>
		<th>ID</th>
		<th>Name</th>
	</tr>
	<? this.items.forEach(function(item){ ?> <!-- LOOP START -->
		<tr>
			<td>{{var item.id}}</td>
			<td>{{var item.name}}</td>
		</tr>
	<? }); ?> <!-- LOOP END -->
</table>
